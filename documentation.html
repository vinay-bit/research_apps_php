System Overview

Research Apps is a comprehensive management system designed to streamline research project workflows, student management, and publication tracking. The system provides tools for managing the entire research lifecycle from student enrollment to publication submission.

Student Management

Track student information, applications, counselor assignments, and academic progress throughout their research journey.

Project Management

Organize research projects, assign mentors and students, track progress, and manage project deliverables.

Publication Workflow

Manage the publication process from draft submission to conference/journal applications and final publication.

User Management

Manage different user roles including administrators, mentors, counselors, and RBMs with appropriate permissions.

Getting Started

Accessing the System

  1. Open your web browser and navigate to the Research Apps URL
  2. You'll be redirected to the login page if not authenticated
  3. Enter your username and password provided by your administrator
  4. Click the "Login" button to access the system
Tip: If you forget your password, contact your system administrator for password reset assistance.

First Login

Upon your first login, you'll see the main dashboard with navigation options based on your user role:

  • Top Navigation: Quick access to main modules
  • Sidebar: Detailed navigation with sub-modules
  • Dashboard: Overview statistics and recent activities
  • User Menu: Profile settings and logout option

Dashboard Overview

The dashboard serves as your central hub, providing system statistics and quick access to frequently used functions.

Statistics Cards

Quick view of total students, projects, publications, and users in the system.

Recent Activities

Timeline of recent system activities including new registrations and project updates.

Quick Actions

Shortcut buttons to common tasks like adding new students or creating projects.

System Alerts

Important notifications about deadlines, pending approvals, or system updates.

Student Management

The Student Management module allows you to maintain comprehensive records of all students participating in research programs.

Adding New Students

  1. Navigate to Student Management â†’ Add Student
  2. Fill in required information: Full Name, Email, Contact, Affiliation, Grade, Application Year
  3. Assign counselor and RBM from dropdown menus
  4. Select appropriate board if applicable
  5. Click Save Student to create the record
Tip: Student IDs are automatically generated in the format STU2025XXXX. You don't need to enter this manually.

Student Search and Filtering

The student list provides powerful search and filtering capabilities:

Search Options:

  • Text Search: Search by name, student ID, or email
  • Filter by RBM: View students assigned to specific RBMs
  • Filter by Counselor: View students under specific counselors
  • Filter by Year: View students by application year

Using Search and Filters:

  1. Go to Student Management â†’ Student List
  2. Use the search box to enter keywords
  3. Select filters from dropdown menus
  4. Click Search to apply filters
  5. Click Clear to reset all filters

Editing Student Information

  1. Find the student in the student list
  2. Click the Edit button in the Actions column
  3. Modify the necessary information
  4. Click Update Student to save changes
Warning: Only administrators can delete student records. Deletion is permanent and will remove all associated project assignments.

Project Management

The Project Management module handles the entire lifecycle of research projects from creation to completion.

Creating New Projects

  1. Navigate to Project Management â†’ Create Project
  2. Enter project details: Name, Description, Subject, Dates, Drive Link, Notes
  3. Set project status from predefined options
  4. Assign lead mentor from available mentors
  5. Select RBM for regional oversight
  6. Assign students to the project
  7. Add relevant tags for categorization
  8. Click Create Project to save
Tip: Project IDs are automatically generated in PRJ2025XXXX format. End dates are automatically set to 4 months from start date if not specified.

Project Status Management

Projects progress through predefined statuses:

  • Project Execution - yet to start: Initial planning phase
  • Project Execution - in progress: Active development
  • Project Execution - completed: Development finished
  • Research Paper - in progress: Writing phase
  • Research Paper - completed: Ready for publication

Managing Project Assignments

Assigning Students:

  1. Open the project for editing
  2. In the "Assigned Students" section, click Select Students
  3. Choose multiple students from the available list
  4. Students are automatically assigned the "Team Member" role
  5. Save the project to confirm assignments

Managing Project Tags:

Tags help categorize and filter projects:

  • Select from existing tags or create new ones
  • Common tags: Research, Innovation, Technology, Science
  • Use tags for filtering in the project list

Publications Management

The Publications module manages the complete publication workflow from initial submission to final publication.

Ready for Publication

Projects that have completed research and are ready for paper submission.

In Publication

Papers currently being reviewed or in the publication process.

Conference Applications

Track submissions to various conferences with deadlines and status.

Journal Submissions

Manage journal publication submissions and review processes.

Managing Ready for Publication

  1. Navigate to Publications â†’ Ready for Publication
  2. Click Add New Publication
  3. Select the completed project from dropdown
  4. Enter publication details: Paper Title, Abstract, Keywords, Domain
  5. Upload documents: First Draft, Plagiarism Report, Final Paper
  6. Assign students involved in the publication
  7. Save the publication record

Conference and Journal Management

Adding Conferences:

  1. Go to Publications â†’ Conferences
  2. Click Add Conference
  3. Enter conference information: Name, Deadlines, Location, Website
  4. Set the submission status and save
Tip: Use the deadline tracking feature to get alerts about upcoming submission deadlines.

User Management

User Management allows administrators to create and manage user accounts with appropriate roles and permissions.

Creating New Users

  1. Navigate to User Management â†’ Create User
  2. Select user type: Admin, Mentor, Counselor, RBM
  3. Enter personal information: Name, Username, Password, Email
  4. Add role-specific information: Department, Specialization, Branch
  5. Set account status (Active/Inactive)
  6. Click Create User to save

Common Workflows

Complete Student Research Journey

  1. Student Registration: Add student with basic info, assign counselor and RBM
  2. Project Assignment: Create project, assign mentor and students, set timeline
  3. Project Execution: Track progress, update status, manage team assignments
  4. Publication Process: Move to "Ready for Publication", upload papers, submit to venues

Project Status Updates

  1. Review all active projects weekly
  2. Update project status based on current progress
  3. Add notes about challenges or achievements
  4. Adjust timelines if necessary
  5. Communicate updates to stakeholders

Frequently Asked Questions

Common Issues

Login Problems: Verify username and password. Contact administrator if account is locked.
Permission Errors: Check your user role permissions. Contact administrator for additional access.

Questions & Answers

Q: How do I change my password?

Contact your system administrator for password changes and resets.

Q: Can students be assigned to multiple projects?

Yes, students can be assigned to multiple projects simultaneously.

Q: How are IDs generated?

IDs are automatically generated: STU2025XXXX for students, PRJ2025XXXX for projects.
